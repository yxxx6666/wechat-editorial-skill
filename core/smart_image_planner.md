# Smart Image Planner v0.3.5

目标：根据文章内容自动规划公众号封面图和正文配图，而不是固定套用单一视觉模板。

## 默认流程

1. 识别内容类型、情绪、场景和传播目标。
2. 匹配视觉风格。
3. 生成 2.35:1 横版封面提示词，并兼容中心 1:1 裁切。
4. 根据文章长度和章节结构规划 1-3 张 3:4 正文配图。
5. 输出图片任务清单、插入位置和质检规则。
6. 生图执行保持可选，默认 prompt_plan_first。

## 风格匹配

- AI / 科技 / 商业：现代科技知识杂志风。
- 健康 / 生活方式：温暖真实生活感。
- 宇宙 / 硬核科普：深色电影感宇宙科普风。
- 个人成长：温暖克制个人成长风。
- 通用知识文：清爽知识杂志风。

## 版权安全

必须加入背景重构法去水印 / 水印处理：仅限自有图或本流程生成图；第三方版权水印不得自动移除。发现水印时默认重新生成或更换素材。只允许对自有图或本流程生成图做背景延展、边缘补全和瑕疵修复。
