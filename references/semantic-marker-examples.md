# 语义标记示例

- `关键变量是反馈速度。` → 主题色下划线。
- `过去常说只要更努力就行。` → 灰色删除线。
- `不要把所有提醒都开着。` → 红色下划线或风险色块。
- `建议先关闭非必要通知。` → 绿色段内高亮。
- `每天 3 小时，一年 1000 多小时。` → 数字强调。
- `你以为问题是时间不够。真正的问题是注意力被切碎了。` → 双栏对比。
- `编辑注：这里说的不是不努力。` → 批注式旁白。
- 三个原文列表项 → `01/02/03` 与内容绑定的编号卡。
- 完整引语 → 左侧竖线引用，成对引号保持在同一组件。

禁止：同一段同时出现删除线、下划线和高亮；同一篇文章连续堆叠多张渐变卡；为制造样式擅自补写反面观点。
