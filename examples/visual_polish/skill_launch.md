发布一个公众号排版 Skill
我做这个 Skill，是因为很多文章不是内容不好，而是复制到公众号之后变得很散。
它要解决的问题很明确：把普通 Markdown 文章转成公众号安全 HTML，同时保留杂志感和阅读节奏。
核心能力包括文章类型识别、段落节奏规划、视觉组件选择、图注绑定、表格降级和 release 校验。
适合写公众号、课程说明、知识科普、Skill 发布和读书笔记的人使用。
使用方法很简单，准备一篇 Markdown，运行转换脚本，再检查生成的 HTML 和 validation report。
项目地址可以放在文末说明区，不要做成强按钮。
下一步我会继续打磨个人公众号风格系统，但这一版只先把排版研究落地。
