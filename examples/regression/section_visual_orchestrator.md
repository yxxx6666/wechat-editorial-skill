# 章节视觉编排回归测试

01
为什么比较会让生活失去边界

过去，一个人主要和身边的人比较。

现在，你看到的是整个世界的极端样本。

学校比较升学率。
公司比较增长率。
运动员比较数据。
创作者比较播放量。

02
为什么注意力需要主动防守

注意力已经变成一种需要主动防守的资源。

你正在吃饭，却可以看到战争、股市和陌生人的愤怒。

03
怎样重新为生活建造房间

这件事和我的现实生活有关吗？
我能采取什么具体行动吗？
看完之后，我会变得更清醒，还是只会更愤怒？

少看所有人的排行榜，多记录自己的长期变化。
用主动选择的订阅，替代永远刷不完的推荐。

一个人真正需要的，也许不是更强的自控力，而是重新拥有几个不会被整个世界闯进来的房间。
