# 真正拖垮你的，可能不是时间不够
你以为问题是时间不够。真正的问题是注意力被切碎了。
## 先修正旧认知
过去常说只要更努力就能解决问题。真正关键的不是更努力，而是换掉错误的反馈系统。
每天 3 小时碎片化刷手机，一年就是 1000 多小时。
## 再看风险和行动
不要把所有提醒都开着，它会持续切断注意力。
建议先关闭非必要通知，再给重要任务设置一个完整时间块。
编辑注：这里说的不是不努力，而是先减少无效消耗。
## 把方法拆成三步
- 先减少输入噪音
- 再建立反馈系统
- 最后固定复盘节奏
“真正拖垮人的，不是任务多，而是每个任务都没有边界。”
## 最后回到环境
重点不是你有没有自律。重点是你的环境有没有帮你减少选择成本。
